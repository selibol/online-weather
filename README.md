# online-weather

## create virtualenv

  `python -m venv venv`
  
## activate venv

 `venv\Scripts\activate`

  
 ## install the packages
 
 `pip install -r requirements.txt`
 
 ## create user table
 
 `python manage.py migrate`
 
 ## create user
 
  `python manage.py createsuperuser`
  
  ## run project
  
   `python manage.py runserver`
